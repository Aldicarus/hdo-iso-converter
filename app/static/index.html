⚙︎ Configuración

      
API keys e integraciones. Se persisten ofuscadas en local; ganan sobre las variables de entorno.

    


    

    

      

        🎬 TMDb — Clave de la API
        
      

      

        Habilita la traducción de títulos ES→EN en la recomendación CMv4.0 (Tab 3). Sin Clave de la API, el match usa el nombre crudo del fichero — peor cobertura con traducciones agresivas (ej. Jungla de cristal → Die Hard).
        
Obtener gratis en themoviedb.org/settings/api → campo “Clave de la API” (v3 auth).
      

      

        
        
      

      

    


    

      

        📦 Google API key (Drive + Sheets)
        
      

      

        Habilita (1) listado y descarga de RPUs del repositorio DoviTools en Google Drive, y (2) lectura de la hoja de recomendaciones con los hyperlinks originales (imágenes HDR COMP, gráficos, comparativas) incrustados en las celdas.
        
Crear en console.cloud.google.com/apis/credentials → Crear credenciales → Clave de API. Habilitar dos APIs en la biblioteca: Google Drive API y Google Sheets API.
      

      

        
        
      

      

    


    

      

        🎯 URL del repositorio DoviTools
        
      

      

        Google Drive folder con los bins .bin CMv4.0 de REC_9999. El acceso es privado — se obtiene haciendo una donación al autor (15 CAD$) en este link
        PayPal · REC_9999. Al donar, indica el correo y pide acceso al repositorio de RPUs. Recibirás el link de la carpeta de Google Drive; pégalo aquí.
        
Hasta configurar esto, toda la sección "Repo DoviTools" queda deshabilitada.
